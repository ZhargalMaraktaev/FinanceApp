Инструкция по пользованию приложением "Учет финансов"  

1. Установка и запуск  
   Убедитесь, что у вас установлена Visual Studio (версия 2022 или выше) с поддержкой .NET (SDK версии 6.0 или выше).  
   Откройте проект в Visual Studio:  
   - Скачайте проект или клонируйте его из репозитория:  
     git clone https://github.com/yourusername/FinanceApp.git  
   - Откройте проект в Visual Studio:  
     Файл -> Открыть -> Проект/Решение -> Выберите файл FinanceApp.sln  
   - Запустите проект:  
     Нажмите клавишу F5 или выберите в меню: Отладка -> Начать отладку.  

2. Главное меню  
   После запуска вы увидите главное меню, где можно выбрать действие:  
   Выберите действие:  
   1. Добавить категорию  
   2. Добавить операцию  
   3. Показать аналитику  
   4. Экспортировать данные в JSON  
   5. Выйти  
   Ваш выбор:  

3. Добавление категории  
   В главном меню выберите 1. Добавить категорию.  
   Введите данные:  
   - ID категории: Уникальный числовой идентификатор категории.  
   - Тип категории: Доходы (Income) или расходы (Expense).  
   - Название категории: Краткое описание категории (например, "Зарплата" или "Кафе").  
   Пример ввода:  
   Введите ID категории: 1  
   Введите тип категории (Income/Expense): Income  
   Введите название категории: Зарплата  
   Категория будет добавлена, и вы увидите сообщение:  
   Command executed in 5 ms  

4. Добавление операции  
   В главном меню выберите 2. Добавить операцию.  
   Введите данные:  
   - ID операции: Уникальный числовой идентификатор операции.  
   - Тип операции: Доходы (Income) или расходы (Expense).  
   - ID счета: Идентификатор банковского счета (например, 1).  
   - Сумма: Сумма операции (для расходов введите отрицательное число, например, -50).  
   - Описание: Краткое описание операции.  
   - ID категории: Идентификатор категории, к которой относится операция.  
   Пример ввода:  
   Введите ID операции: 1  
   Введите тип операции (Income/Expense): Income  
   Введите ID счета: 1  
   Введите сумму: 1000  
   Введите описание: Зарплата за октябрь  
   Введите ID категории: 1  
   Операция будет добавлена, и вы увидите сообщение:  
   Операция добавлена.  

5. Просмотр аналитики  
   В главном меню выберите 3. Показать аналитику.  
   Введите данные:  
   - Начальная дата: Дата начала периода в формате yyyy-MM-dd.  
   - Конечная дата: Дата конца периода в формате yyyy-MM-dd.  
   Пример ввода:  
   Введите начальную дату (yyyy-MM-dd): 2023-10-01  
   Введите конечную дату (yyyy-MM-dd): 2023-10-31  
   Вы увидите результаты:  
   - Чистый доход: Разница между доходами и расходами за выбранный период.  
   - Группировка по категориям: Сумма операций по каждой категории.  
   Пример вывода:  
   Чистый доход: 950  
   Категория 1: 950  

6. Экспорт данных в JSON  
   В главном меню выберите 4. Экспортировать данные в JSON.  
   Данные будут экспортированы в файл exported_data.json в папке bin/Debug/net6.0 проекта.  
   Вы увидите сообщение:  
   Данные экспортированы в файл exported_data.json  

7. Выход из приложения  
   В главном меню выберите 5. Выйти.  
   Приложение завершит работу.

    "Type": "Income",

    "Amount": 1000,  
    "Date": "2023-10-01T00:00:00",  
    "Description": "Зарплата за октябрь",  
    "CategoryId": 1  
  }  
]
